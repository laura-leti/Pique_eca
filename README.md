# Pique_eca

MINI-GAME — Maio Laranja (Pique ECA)

O projeto será desenvolvido em Python com versionamento pelo Git/GitHub. A proposta consiste em um mini-game interativo voltado para conscientização sobre o Maio Laranja e prevenção ao abuso infantil.

Funcionamento do sistema

Ao iniciar o programa, o usuário deverá informar:

Nome
Idade

Como funcionalidade adicional, o sistema poderá questionar se o jogador possui filhos ou convive com crianças em seu meio social.

Estrutura do jogo

Após o cadastro inicial, o jogador será direcionado para perguntas de conscientização relacionadas à sua faixa etária/grupo social.

As perguntas serão divididas em:

Crianças/adolescentes
Adultos/responsáveis

Cada grupo terá aproximadamente 9 perguntas, separadas por níveis de dificuldade:

Fácil
Difícil
Muito difícil
Sistema de pontuação

Cada resposta correta gerará pontos para o jogador.

Ao final:

A pontuação será somada
O jogador visualizará sua colocação
O sistema atualizará um ranking em tempo real para acompanhamento durante a apresentação (Tamâra irá ajudar a desenvolvê-lo na aula de quarta-feira).
Ranking

O ranking exibirá:

Nome do jogador
Pontuação
Colocação

A atualização será automática conforme os jogadores finalizarem o mini-game.

Objetivo do projeto

O principal objetivo é promover conscientização sobre prevenção, proteção infantil e identificação de situações de risco, utilizando o game como ferramenta educativa e interativa.
